Project Overview

Project Overview

The Red Yes Studio Website is a bold, rich, and 'punchy' website designed for a theatre production studio. The website focuses on clean typography and abstract/ethereal background imagery that conveys the diversity of work across different theatre productions the studio has developed.

Design Approach

The layout is very asymmetric, allowing the user to flow down the page in a 'dreamlike' state. This unique approach reflects the creative and artistic nature of theatrical work while maintaining professional functionality and responsive design.
